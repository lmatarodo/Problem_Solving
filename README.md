# Baekjun

백준 온라인 저지 문제풀이를 위한 디렉토리입니다.

## 목표
- 하루에 한 번 백준 문제를 풀어서 알고리즘 실력을 향상시키기
- PS(Problem Solving) 능력 개발
- 다양한 알고리즘과 자료구조 학습

## 구조
```
Baekjun/
├── README.md          # 이 파일
├── hello.c           # 첫 번째 C 프로그램
├── .vscode/          # VS Code 설정
└── output/           # 컴파일된 실행 파일들
```

## 사용 언어
- C (주요 언어)
- 향후 필요에 따라 다른 언어 추가 예정
